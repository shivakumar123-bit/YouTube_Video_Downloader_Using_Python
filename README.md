# YouTube_Video_Downloader_Using_Python

A small Python utility to download WebM video formats from YouTube with a Tkinter folder picker.

## Requirements
- Python 3.8+
- yt-dlp (`pip install -r requirements.txt`)
- FFmpeg (must be in PATH) for merging audio/video
- On Linux: `python3-tk` package may be required for Tkinter UI

## Step-by-step Process
- step1 : Open VScode.
- step2 : Run the code in VScode Terminal.
- step3 : Provide the URL.
- step4 : Select the Path to save the video.
- step5 : Select the Quality of the video.
- step6 : Wait until the completion of video download.
- step7 : Select Ok after the Completion.
